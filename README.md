# DREAMBRIGHT Event Managements

Deploy-ready React + Tailwind frontend.

## Run locally

```powershell
npm config set registry https://registry.npmjs.org/
npm install
npm run dev
```

## Home hero slideshow

The Home hero uses event images from dreambright.in and changes every 2 seconds.
If the old version appears in the browser, stop Vite, run `npm run dev` again, and hard refresh with `Ctrl + Shift + R`.
